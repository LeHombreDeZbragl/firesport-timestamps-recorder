# Video Timestamp Recorder - Complete Guide (Video Only)

## 🔇 **Audio-Free Design**
This video player is designed for **video-only operation** with all audio functionality removed for maximum stability and performance. Focus on visual content and precise timestamp recording without audio distractions.

## 🎬 Segment Creation & Naming

### 📝 **Naming Segments**
- **Segment Name Input**: Type custom names directly in the input field
- **Auto-naming**: Leave input empty for automatic "Segment 1", "Segment 2", etc.
- **Smart Suggestions**: Placeholder text shows next suggested name
- **Focus on Start**: Input field automatically gets focus when you save a start timestamp

### ⌨️ **Naming Shortcuts**
- **Enter** - Save end timestamp with current name in input field
- **N** - Focus the segment name input field and select existing text
- **E** - Save end timestamp (same as Enter when input is focused)

### 🎯 **Naming Workflow**
1. **Press S** - Save start timestamp (input field gets focus)
2. **Type name** - Enter your custom segment name (optional)
3. **Press Enter or E** - Save end timestamp with the name
4. **Field clears** - Ready for next segment

## 🎮 Frame-by-Frame Navigation

### 🎮 Button Controls
- **⏪ Frame Back** - Move one frame backward
- **⏩ Frame Forward** - Move one frame forward

### ⌨️ Keyboard Shortcuts

#### Single Frame Navigation
- **Comma (,)** - Move one frame backward
- **Period (.)** - Move one frame forward
- **Left Arrow** - Move one frame backward (alternative)
- **Right Arrow** - Move one frame forward (alternative)

#### Fast Navigation (10 frames)
- **Shift + Left Arrow** - Move 10 frames backward
- **Shift + Right Arrow** - Move 10 frames forward

#### Other Controls
- **S** - Save start timestamp
- **E** - Save end timestamp
- **N** - Focus segment name input
- **Space** - Play/Pause toggle

### 🎯 Features
- **Smart Frame Detection**: Automatically detects video frame rate for precise navigation
- **Auto-Pause**: Automatically pauses video when using frame navigation for precision
- **Boundary Protection**: Prevents seeking beyond video start/end
- **Fallback Frame Rate**: Uses 30fps default if frame rate cannot be detected

### 💡 Tips
1. **Load a video first** to enable all controls
2. **Use tooltips** - hover over buttons for detailed information
3. **Quick naming** - press N to quickly focus and rename segments
4. **Enter key** - works in the name input to save segments quickly
5. **Auto-suggestions** - placeholder text shows suggested names

### 🎬 Complete Workflow Example
1. **Load video** - click "📂 Load Video" or drag & drop
2. **Navigate** - use Space for rough positioning, arrows for precision
3. **Start segment** - press S to mark beginning
4. **Name it** - type custom name (input gets focus automatically)
5. **End segment** - press Enter or E to save with timestamp
6. **Repeat** - field clears and suggests next segment name
7. **Edit anytime** - modify segments directly in the text area
8. **Export** - save timestamps to file when done

### 📋 Text Editing
- **Direct editing** - modify segment names and timestamps in the text area
- **Format**: `Number. Title` followed by `HH:MM:SS.mmm - HH:MM:SS.mmm`
- **Live parsing** - changes are applied automatically as you type
- **Delete segments** - remove lines to delete segments
